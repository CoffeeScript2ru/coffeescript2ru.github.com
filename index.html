Начало

CoffeeScript — язык программирования, транслируемый в JavaScript. CoffeeScript добавляет синтаксический сахар в духе Ruby, Python и Haskell для того, чтобы улучшить читаемость кода и уменьшить его размер. В среднем для выполнения одинаковых действий на CoffeeScript требуется в 2 раза меньше кода, чем на JavaScript. Автором является Jeremy Ashkenas.

Язык может компилироваться в JS на серверной стороне - для этого необходимо установить Node.js. Так же возможна компиляция на браузерной стороне - для этого необходимо подключить JS компилятор.

Попробовать CoffeeScript можно вживую на основном сайте www.coffeescript.org, нажав на "TRY"
